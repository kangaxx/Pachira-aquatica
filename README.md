# Pachira-aquatica

这是一个基于Spring Boot的基础项目。

## 项目说明

本项目是一个简单的Spring Boot应用，包含了基本的Web功能，可以作为开发的起点。

## 功能特性

- Spring Boot 3.5.7
- Java 17
- REST API支持
- Maven构建

## 快速开始

### 环境要求

- JDK 17或更高版本
- Maven 3.6或更高版本

### 构建和运行

```bash
# 构建项目
./mvnw clean package

# 运行应用
./mvnw spring-boot:run
```

### 访问API

启动应用后，可以通过以下URL访问示例API：

```
http://localhost:8080/hello
```

## 项目结构

- `src/main/java` - 源代码目录
- `src/main/resources` - 资源文件目录
- `src/test` - 测试代码目录
